## Spring Boot + JUnit 5 + Mockito

Dans ce repo, nous allons voir comment effectuer le test d'intégration de Spring Boot 3 avec JUnit 5, ainsi que Mockito.

### Technologies
---

- Spring Boot 3
- JUnit 5 Framework
- Mockito
- Maven

### Exemples
---

- Test service
- Test service Mock
- Test controller

### Exécuter les test
---
`$ mvn test`
